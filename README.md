# Streaming Refactor - ERP de Streaming (Monorepo)

Este repositorio contiene la plataforma tipo ERP orientada a la gestión integral de un negocio de reventa de cuentas de streaming.

## Estructura del Repositorio

- `/backend`: API REST asíncrona desarrollada en Python con FastAPI, SQLAlchemy 2.0 y PostgreSQL (Supabase).
- `/frontend`: Aplicación web responsive (Mobile First) desarrollada en React con Vite, TypeScript y Tailwind CSS.
- `/docs`: Documentación del proyecto, incluyendo lógica de negocio y plan de desarrollo.

## Requisitos Previos

- Python 3.10+
- Node.js 18+
- Base de datos PostgreSQL (Supabase)
